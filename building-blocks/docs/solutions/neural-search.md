# Neural Search

Neural Search is a solution for semantic search and retrieval using neural networks and vector embeddings to understand and match content.

## Key Capabilities Used

- [Similarity Search](../capabilities/similarity-search)
- [Multi-Modal Understanding](../capabilities/multi-modal-understanding)

## Features

- Semantic matching
- Cross-modal search
- Vector indexing
- Query understanding
- Relevance ranking

## Use Cases

- Document search
- Image retrieval
- Code search
- Product discovery
- Knowledge base search

## Technologies

- Neural embeddings
- Transformer models
- ANN algorithms
- Vector quantization
- Cross-encoders
